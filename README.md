# SheetMerge
Sheet merging tool for Google Sheets. A free webapp version of this script can be found [here](https://script.google.com/macros/s/AKfycbz2Do4_UD8MvRfamN9RdcaTn3Gcwsc9m6IeWOkFQr70KEu3lAOqLHWC8sbR5GvT3x_Vqw/exec) (requires google account).

## Installation
1. Open any Google Sheet.
2. Go to Extensions > Apps Script.
3. Create the Code.gs and Dialog.html files. Paste the contents of their equivalents here into those files.
4. Save both files (Ctrl+S or Cmd+S).
5. Refresh your Google Sheet. You should see a new "Merge Tool" menu appear in the toolbar.
6. Click Merge Tool > Open Merge Dialog to use it. Authorise the script as normal.
